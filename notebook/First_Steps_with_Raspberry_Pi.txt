===== Initial Setup Steps =====
* Flash an image, using the flasher. Set connection details for a known SSID.
* Start device
* Use nmap to locate device on network.
* Log into device.
* Issue following commands

===== Network Configuration =====
==== WiFi Connections ====
Will connect to iotrac SSID automatically.
IP: 192.168.0.2/24
Username: pi
Key access only.

Will connect to internet SSID automatically
IP: 192.168.1.48

==== Add WIFI connection ====
{{{code: lang="sh" linenumbers="True"
# Add the connection
sudo nmcli con add type wifi ifname wlan0 con-name "iotrac" ssid "<iotrac>"

# Set the password
sudo nmcli con modify "iotrac" wifi-sec.key-mgmt wpa-psk wifi-sec.psk "<password>"
}}}

==== Static config for iotrac SSID ====
This will configure a static ip when connecting to iotrac
{{{code: lang="sh" linenumbers="True"
# Set static IP configuration
sudo nmcli con modify "iotrac" ipv4.addresses "192.168.0.2/24"
sudo nmcli con modify "iotrac" ipv4.gateway "192.168.0.1"
sudo nmcli con modify "iotrac" ipv4.dns "192.168.0.1,8.8.8.8"
sudo nmcli con modify "iotrac" ipv4.method manual
sudo nmcli con modify "iotrac" connection.autoconnect yes
}}}

==== Delete pre-configured ====
{{{code: lang="sh" linenumbers="True"
sudo nmcli con delete "preconfigured"
}}}

===== USB Configuration =====
IP: 192.168.7.2

You will need to configure your host's interface
{{{code: lang="sh" linenumbers="False"
ip link # This will show you a list of interface. The PI is likely on the last
ip addr add 192.168.7.1/24 dev <iface>
}}}

==== Enable USB ====
Issue these commands on the PI, after connecting to it through WIFI
{{{code: lang="sh" linenumbers="True"
# Create a connection for the USB interface
sudo nmcli con add type ethernet ifname usb0 con-name "usb-gadget"

# Set a static IP for the Pi side
sudo nmcli con modify "usb-gadget" ipv4.addresses "192.168.7.2/24"
sudo nmcli con modify "usb-gadget" connection.autoconnect yes

# Set static IP method
sudo nmcli con modify "usb-gadget" ipv4.method manual
}}}

==== Create systemd unit that makes sure USB interface comes up ====
{{{code: lang="sh" linenumbers="True"
sudo nano /etc/systemd/system/usb-gadget-autoconnect.service
}}}

Put this:
{{{code: lang="sh" linenumbers="True"
[Unit]
Description=Auto-activate USB gadget network
After=network.target
Wants=network.target

[Service]
Type=oneshot
ExecStart=/usr/bin/nmcli con up "usb-gadget"
RemainAfterExit=yes

[Install]
WantedBy=multi-user.target
}}}

Enable unit
{{{code: lang="sh" linenumbers="True"
sudo systemctl enable usb-gadget-autoconnect.service
sudo systemctl daemon-reload
}}}

===== Challenges & Solutions =====
==== How to connect to the PI using wifi ====
==== Possible solutions ====
==== Solution 1 ====
* Have a phone, with a fixed SSID.
* Pi connects to SSID. 
* Controller laptop connects to SSID.

==== Limitations ====
* Works with DHCP.
* IP of PI is unknown.

==== Current Solution ====
* Static IP configuration implemented (192.168.0.2/24 for iotrac)
* USB gadget interface as backup connection (192.168.7.2)
* SSH key authentication configured
* Multiple connection methods available

===== SSH Configuration File =====
To access and edit SSH configuration file:
{{{code: lang="sh" linenumbers="True"
vim ~/.ssh/config
}}}

==== Current SSH Config ====
{{{code: lang="sh" linenumbers="True"
# Raspberry Pi iotrac connection
Host raspi
    Hostname 192.168.0.2
    User pi
    IdentityFile ~/.ssh/id_rsa

# Raspberry Pi internet connection
Host raspi_internet
    Hostname 192.168.1.48
    User pi
    IdentityFile ~/.ssh/id_rsa

# Raspberry Pi USB connection
Host raspi_usb
    Hostname 192.168.7.2
    User pi
    IdentityFile ~/.ssh/id_rsa
}}}

==== Connection Commands ====
{{{code: lang="sh" linenumbers="True"
# Connect via iotrac WiFi
ssh raspi

# Connect via internet WiFi
ssh raspi_internet

# Connect via USB
ssh raspi_usb
}}}

===== WiFi Cellphone Backup Configuration =====
==== Add Cellphone WiFi Connection ====
{{{code: lang="sh" linenumbers="True"
# Add cellphone WiFi connection
sudo nmcli con add type wifi ifname wlan0 con-name "cellphone" ssid "<cellphone_ssid>"

# Set password for cellphone WiFi
sudo nmcli con modify "cellphone" wifi-sec.key-mgmt wpa-psk wifi-sec.psk "<cellphone_password>"

# Set static IP for cellphone connection
sudo nmcli con modify "cellphone" ipv4.addresses "192.168.43.100/24"
sudo nmcli con modify "cellphone" ipv4.gateway "192.168.43.1"
sudo nmcli con modify "cellphone" ipv4.dns "192.168.43.1,8.8.8.8"
sudo nmcli con modify "cellphone" ipv4.method manual
sudo nmcli con modify "cellphone" connection.autoconnect yes
}}}

==== Update SSH Config for Cellphone ====
Add to ~/.ssh/config:
{{{code: lang="sh" linenumbers="True"
# Raspberry Pi cellphone connection
Host raspi_cellphone
    Hostname 192.168.43.100
    User pi
    IdentityFile ~/.ssh/id_rsa
}}}

==== Connection Priority ====
1. **iotrac** (192.168.0.2) - Primary, no internet
2. **cellphone** (192.168.43.100) - Backup with internet
3. **internet** (192.168.1.48) - Secondary WiFi
4. **usb** (192.168.7.2) - Direct connection backup
